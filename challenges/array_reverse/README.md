# README

## Perfect Sequence

### Author: Alex Angelico, Anthony Beaver

### Problem Domain

Given a list, return that list with the items in reversed order.

### Inputs and Expected Outputs

Input |	Expected Output
----- | ---------------
[1, 2, 3] |	[3, 2, 1]
[4, 5, 6] | [6, 5, 4]

### Big O
Time: O(N)
Space: O(N)
