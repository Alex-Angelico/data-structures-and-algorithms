# README

## Multiple Binary Tree Common Values

### Author: Alex Angelico

### Problem Domain

Create a function that will generate a list of common values in two binary trees.

### Inputs and Expected Outputs

Input | Expected Output
----- | ---------------
{2} <---> {4} <---> {5} |
{1} <---> {2} <---> {4} | [2, 4]

### Big O

Time: O(log(n))  
Space: O(n)
