# README

## Hash Table Implementation

### Author: Alex Angelico

### Problem Domain

Create a Hashtable class with _hash, set, get, and contains methods.

### Inputs and Expected Outputs

Input | Expected Output
----- | ---------------
[0, 1, 2, 3, 4], {1: ab}, {4: cd} | [0, 1, 2, 3, 4]
                                     {1: ab}  {4: cd}

### Big O

Time: O(1)  
Space: O(1)
